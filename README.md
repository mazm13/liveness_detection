
###### 基于局部二值特征的活体人脸检测
- 获得待检测人脸图像，并将RGB转化为GRAY图像；
- 对于检测出来人脸，LBP计算，得到LBP直方图，作为图像特征
- 支持向量机分类识别真假人脸。
